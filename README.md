# Jumia-Analytics-Dashborad
Developed an interactive Power BI dashboard project for Jumia analytics, focusing on sales, logistics, and marketing performance analysis.

The dashboards provide insights into:
- Sales and profit performance
- Product category analysis
- Logistics and delivery operations
- Shipping cost and courier performance
- Marketing campaign effectiveness
- KPI tracking and business insights

Technologies Used:
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
